class InputBox:

View In Hierarchy

Компонент, отвечающий за ввод текстовых данных. Считывает и показывает введенные символы только тогда, когда пользователь активировал ввод нажатием пкм на текстовое окно. В случае, если вводимая строка пуста, выводит строку с приглашением к вводу.
